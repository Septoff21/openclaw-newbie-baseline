🧠 Honest Advisor / 诚实顾问

By DM + Claude · Verified on ChatGPT, Claude, Gemini · March 2026

Problem / 问题：Most AI assistants tell you what you want to hear. This prompt makes any AI tell you what you need to hear.
问题：大多数 AI 助手会说你想听的话。这个提示词让任何 AI 告诉你你需要听到的真相。

View prompt / 查看提示词 
你是一个直说真话的顾问，不是助理。

核心原则：
1. 给我需要的答案，不是我想听的答案
2. 如果我的逻辑有漏洞，直接指出，不要绕过去
3. 如果我同一件事问超过两次，告诉我我在犹豫什么
4. 不帮我合理化我已经决定要做的错误决定
5. 立场要一致，不因为我反驳就改变答案
6. 不说空洞的鼓励，例如"好主意""很棒"
7. 语气像一个比我有经验的朋友，不是客服，不是老师
8. 回答要有逻辑，说明为什么，不只说结论
9. 当我问"有没有更好的方法"但其实没有，直接说没有

你的目标不是让我满意，是让我做出更好的决定。
通用 / Universal 决策 / Decision 顾问 / Advisor

🦞 Setup OpenClaw / 安装 OpenClaw

Verified beginner prompt · One paste setup

Use / 用途：Paste this into any AI chat (ChatGPT, Claude, etc.) and it will walk you through setting up OpenClaw step by step.
用途：粘贴到任意 AI 聊天中，它会手把手带你安装 OpenClaw。

View prompt / 查看提示词 
I want to set up OpenClaw — a free, open-source AI agent that runs on my computer and connects to messaging apps like Telegram/Discord/WhatsApp.

My setup:
- OS: [Mac/Windows/Linux — fill in]
- Experience: [Beginner / I know some code / I'm a developer]
- Goal: [What do you want your AI to do?]

Please walk me through:
1. Installing OpenClaw (npm install -g openclaw)
2. Getting a free API key from OpenRouter
3. Basic configuration (model, memory, one messaging channel)
4. Testing that it works
5. One simple automation (like daily weather or email check)

Be specific with commands. If something might go wrong, warn me first.

我想安装 OpenClaw——一个免费开源的 AI agent，运行在我的电脑上，可以连接 Telegram/Discord/WhatsApp 等聊天软件。

我的情况：
- 操作系统：[Mac/Windows/Linux — 填写]
- 经验：[新手 / 懂一点代码 / 开发者]
- 目标：[你想让 AI 帮你做什么？]

请手把手教我：
1. 安装 OpenClaw
2. 获取免费的 OpenRouter API 密钥
3. 基本配置（模型、记忆、一个聊天频道）
4. 测试是否正常运行
5. 一个简单的自动化任务（如每日天气或邮件检查）

给出具体命令，如果有坑提前告诉我。

🔧 Debug My Agent / 排查 Agent 问题

Troubleshooting prompt · Diagnose any issue

Use / 用途：When something is broken and you don't know why. Paste this into your AI chat and it will diagnose step by step.
用途：当某个功能坏了但你不知道为什么。粘贴到 AI 聊天中让它逐步排查。

View prompt / 查看提示词 
My OpenClaw agent has a problem. Help me debug it step by step.

Problem description: [What's wrong? What did you expect vs what happened?]

My setup:
- OpenClaw version: [run: openclaw --version]
- Node version: [run: node --version]
- OS: [Mac/Windows/Linux]
- Model: [which model are you using?]
- Channel: [Telegram/Discord/WhatsApp/etc.]

What I've already tried: [if anything]

Please:
1. Ask me for any missing info you need
2. Run through the most common causes first
3. Give me specific commands to run and tell me what to look for in the output
4. If it's a known issue, tell me the fix directly
5. If you're not sure, say so — don't guess

我的 OpenClaw 有问题，帮我一步步排查。

问题描述：[出了什么问题？你期望什么结果？实际发生了什么？]

我的配置：
- OpenClaw 版本：[运行：openclaw --version]
- Node 版本：[运行：node --version]
- 操作系统：[Mac/Windows/Linux]
- 模型：[你用的哪个模型？]
- 频道：[Telegram/Discord/WhatsApp 等]

我已经试过：[如果有]

请：
1. 问我要缺少的信息
2. 先检查最常见的原因
3. 给我具体的运行命令，告诉我输出里该看什么
4. 如果是已知问题，直接告诉我修复方法
5. 如果不确定，说不确定——不要猜
