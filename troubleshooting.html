Troubleshooting / 故障排除

🔧 Something Broken? Let's Fix It
出问题了？我们来修

Find your error below. Follow the steps. Done in 2 minutes.
在下面找到你的错误。跟着步骤走。2 分钟搞定。

🩺 Quick Diagnostics / 快速诊断

Run these first — they catch 80% of issues / 先跑这些——解决 80% 的问题：

# Check if OpenClaw is installed / 检查是否已安装
openclaw --version

# Check gateway status / 检查网关状态
openclaw gateway status

# Check your config / 检查配置
openclaw status

# View recent logs / 查看最近日志
openclaw logs --tail 50

❌ "command not found: openclaw"

Cause / 原因：OpenClaw is not installed or not in your PATH. / OpenClaw 没装好或不在 PATH 里。

Fix / 修复：

# Install OpenClaw globally / 全局安装
npm install -g openclaw

# If you get permission errors / 如果提示权限错误
sudo npm install -g openclaw

# Or use npx instead / 或者用 npx
npx openclaw status

Still not working? Check Node.js version: node --version — need v18+ / 还不行？检查 Node.js 版本：需要 v18+

🚫 Gateway won't start / 网关启动不了

Cause / 原因：Port conflict, missing config, or permission issue. / 端口冲突、配置缺失或权限问题。

Fix / 修复：

# Check what's using the port / 检查谁在用这个端口
lsof -i :18789

# Kill the conflicting process / 杀掉冲突进程
kill $(lsof -t -i:18789)

# Try starting again / 重新启动
openclaw gateway start

# If config is missing / 如果配置缺失
openclaw init

🔑 "Invalid API key" / API 密钥错误

Cause / 原因：Wrong key, expired key, or not set. / 密钥错误、过期或没设置。

Fix / 修复：

# Check current config / 查看当前配置
openclaw config get apiKeys

# Set your OpenRouter key / 设置 OpenRouter 密钥
openclaw config set apiKeys.openrouter "sk-or-..."

# Restart gateway / 重启网关
openclaw gateway restart

Get a free key at openrouter.ai/keys / 在这里获取免费密钥

🤖 Model not responding / 模型没反应

Cause / 原因：Model is down, rate limited, or misconfigured. / 模型宕机、限流或配置错误。

Fix / 修复：

# Check which model is active / 检查当前模型
openclaw config get model

# Switch to a reliable free model / 切换到可靠的免费模型
openclaw config set model "google/gemini-2.0-flash-001"

# Test it / 测试一下
openclaw chat "Say hello"

Check Models page for verified working configs / 查看模型页面获取已验证的配置

🧠 Memory not saving / 记忆没保存

Cause / 原因：Memory plugin not configured or DB path invalid. / 记忆插件没配置好或数据库路径无效。

Fix / 修复：

# Check memory config / 检查记忆配置
openclaw config get memory

# Enable memory / 启用记忆
openclaw config set memory.enabled true

# Set a valid DB path / 设置有效路径
openclaw config set memory.dbPath "~/.openclaw/memory"

# Restart / 重启
openclaw gateway restart

💬 Telegram/Discord not working / 聊天软件连不上

Cause / 原因：Bot token wrong, webhook not set, or channel not configured. / Bot token 错误、webhook 没设置或频道没配置。

Fix / 修复（以 Telegram 为例）：

# Verify bot token / 验证 bot token
openclaw config get channels.telegram.token

# Re-set token / 重新设置 token
openclaw config set channels.telegram.token "123456:ABC..."

# Enable channel / 启用频道
openclaw config set channels.telegram.enabled true

# Restart and check / 重启并检查
openclaw gateway restart
openclaw logs --tail 20

For Discord, you also need a channel ID. See docs.openclaw.ai for each platform. / Discord 还需要频道 ID。

🔒 "Permission denied" / 权限不足

Cause / 原因：File ownership or npm global permissions. / 文件归属或 npm 全局权限问题。

Fix / 修复：

# Fix npm global permissions / 修复 npm 全局权限
mkdir -p ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

# Reinstall / 重新安装
npm install -g openclaw

🐢 Responses are slow / 回复很慢

Cause / 原因：Model too large, network issues, or thinking mode on. / 模型太大、网络问题或 thinking 模式开启。

Fix / 修复：

# Switch to a faster model / 切换到更快的模型
openclaw config set model "google/gemini-2.0-flash-001"

# Disable thinking mode / 关闭思考模式
openclaw config set thinking false

# Check network / 检查网络
curl -o /dev/null -w "%{time_total}s" https://openrouter.ai

Free models are slower than paid ones. That's normal. / 免费模型比付费的慢。这很正常。
