# Clone Template Workflow (Beginner)

1. Pick a style card from `style-cards.json`.
2. Copy one prompt from `prompt-library.md`.
3. Generate visuals from `image-prompts.md`.
4. Replace these fields only:
   - name
   - colors
   - logo/brand asset
5. Keep verification checklist unchanged.
6. Publish via GitHub Pages or Vercel.
7. Share final URL.
