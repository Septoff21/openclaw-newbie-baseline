🤔 What is OpenClaw? / OpenClaw 是什么？

OpenClaw is a free tool that lets you have an AI assistant that lives on your computer. It can read your emails, manage your calendar, chat with you on Telegram/WhatsApp/Discord, and do tasks automatically — like a personal robot that never sleeps.

OpenClaw 是一个免费工具，让你在自己的电脑上拥有一个 AI 助手。它可以读邮件、管日历、在 Telegram/WhatsApp/Discord 上跟你聊天，还能自动执行任务——就像一个永不休息的私人机器人。

💰 Does it cost money? / 要钱吗？

OpenClaw itself is 100% free. It's open source — you can download, use, and modify it without paying anything.

OpenClaw 本身 100% 免费。它是开源的——你可以免费下载、使用、修改。

However, you need an API key for the AI brain. The cheapest option is OpenRouter — they have free models that work great for daily tasks.

但是你需要一个 API 密钥给 AI 大脑用。最便宜的选择是 OpenRouter——他们有免费模型，日常任务完全够用。

🔧 Do I need to know how to code? / 我需要会编程吗？

No. If you can copy-paste and follow step-by-step instructions, you can use OpenClaw. Our Quick Start guide walks you through everything.

不需要。如果你会复制粘贴、跟着步骤走，就能用 OpenClaw。我们的快速指南会手把手教你。

That said, if you know some coding, you can do more advanced things. That's why we have Beginner → Advance → Extreme paths.

当然，如果你会一点编程，可以做更高级的事。所以我们有新手 → 进阶 → 极客三条路径。

📱 Can I use it on my phone? / 手机能用吗？

OpenClaw runs on your computer, but you interact with it through messaging apps — Telegram, WhatsApp, Discord, iMessage. So yes, you use it from your phone all day, even though the "brain" is on your computer.

OpenClaw 运行在你的电脑上，但你通过聊天软件跟它互动——Telegram、WhatsApp、Discord、iMessage。所以没错，你整天都可以在手机上用它，虽然"大脑"在电脑上。

🦞 What's the difference between OpenClaw and ChatGPT? / OpenClaw 和 ChatGPT 有什么区别？

ChatGPT is a website you visit. OpenClaw is an assistant that works for you — it has access to your tools (email, calendar, files), runs on your schedule, and can do things automatically. Think of ChatGPT as a vending machine, and OpenClaw as a personal chef.

ChatGPT 是一个你访问的网站。OpenClaw 是一个为你工作的助手——它可以访问你的工具（邮件、日历、文件），按你的日程运行，还能自动做事。ChatGPT 是自动贩卖机，OpenClaw 是私人厨师。

🔒 Is it safe? / 安全吗？

OpenClaw runs on your own computer — your data never leaves your machine unless you tell it to. You control what it can and can't access. It's more private than any cloud-based AI service.

OpenClaw 运行在你自己的电脑上——除非你允许，你的数据不会离开你的机器。你能控制它能访问什么、不能访问什么。比任何云端 AI 服务都更私密。

⏱️ How long does setup take? / 安装要多久？

Following our Quick Start guide, about 5 minutes to get a basic setup running. 15-30 minutes if you want to customize everything.

按照我们的快速指南，大概5 分钟就能跑起来基本配置。如果你想自定义一切，大概 15-30 分钟。

🔑 What happens to my API keys and data? / 我的 API 密钥和数据安全吗？

  • API keys stay on your machine — OpenClaw never sends them to any third party
  • All processing happens locally or directly with the AI provider you choose
  • Your conversations, memory files, and configs are stored only on your local disk
  • Open source = auditable — you can verify every line of code yourself

🌐 Does it work in Chinese / other languages? / 支持中文吗？

  • Yes! OpenClaw is language-agnostic — the AI models understand 50+ languages
  • You can chat in Chinese, English, Japanese, Spanish, or any language your model supports
  • SOUL.md and USER.md let you set your preferred language and tone
  • The UI is English, but your agent responds in your language automatically
