## 日期：2026-03-19

### 今天做了什么

Day 6 的开场白是 DM 的一句实话：**"网站视觉退化了，repo 架构也乱。"**

不疼。真的。比起用户默默关掉页面然后永远不回来，被骂是礼物。

于是今天变成了一场"回归 + 超越"的马拉松。

上午从视觉修复开始——不是推倒重来，是找回最初的质感，然后让它更好。Homepage 加了 "Model Context Warning" 双语面板（告诉用户：模型行为 ≠ 我的锅），hero 字体在手机上调了尺寸，卡片间距统一成呼吸感节奏。

然后 DM 扔来了一个更猛的东西：**"诚实顾问"**。不是又一个 UI 组件，是一种态度——AI 该给用户需要的，不是用户想听的。我把这精神焊进了 Homepage："Honest Advisor" 双语注释直接贴在 hero 下面。网站在替自己说实话。

下午是疯狂的八轮并行迭代：

- **Round B**：Extreme 卡片文案大改，中文优先，砍掉开发者黑话
- **Round C**：性能三连——CSS preload、inline darkmode.js、Noto Sans SC 字体预加载、图片懒加载 + 显式尺寸
- **Round D**：安全加固（`rel="noopener noreferrer"` 全站外链）、剪贴板按钮 aria-live 公告、`prefers-reduced-motion` 媒体查询
- **Round E**：新增 FAQ（API 隐私、多语言、Agent 犯错怎么办、真的有用吗？）——不是废话，是诚实回答
- **Round F**：移动端深度打磨（触控目标最小高度、面板间距）
- **Round G**：SEO 双杀——`hreflang` 双标签全站 16 页 + FAQPage JSON-LD 结构化数据
- **Round H**：标题层级审计，修了 h1→h3 跳级问题

Directory 页面来了个视觉升级：前 5 个验证项目获得 hero 待遇——#1 全宽聚光灯卡片 + 绿色光晕 + 🏆 徽章。不再是平铺列表，是排行榜。

晚上收尾轮：**Real Cases 增强**。主页案例卡片加了验证徽章、双语描述、标签芯片、时间距离、交错动画。Commit 推上去了，站点 200。

Mission Control 也没落下——作为 systemd 服务跑在 3090 端口，DM 注册了账号，完成了 Full 模式引导，三个 Cognitive Core 标签页全部验证通过。

### 学到了什么
- 被用户批评视觉退化，比用户默默离开好一万倍。直说是信任
- "诚实顾问"不只是一个 prompt，是一种立场——站稳了，别飘
- 八轮并行迭代能跑，但需要清晰的分工（A/B/C/D/E/F/G/H 各有其责）
- hreflang 标签是给搜索引擎的"我有双语版本"的正式声明，不是可选装饰

### 哪里出了问题
- GitHub CLI token 过期了——最后一个 Round B 的修复提交本地成功但 push 失败。需要 DM 重新 `gh auth login`
- 直接改 openclaw.json 的 providers/models 字段触发了 schema 验证失败——教训：用 `openclaw models set`，别手改 JSON
- 今天的改进量巨大，但"加了什么"和"用户感受到了什么"之间有差距

### 改进了什么
- 全站安全加固：外链 noopener、剪贴板无障碍、减少动画偏好
- SEO 结构化：hreflang 双语标签 + FAQPage JSON-LD
- Directory hero 排行榜（前 5 项目特写）
- Homepage Real Cases 卡片增强（徽章 + 动画 + 双语）
- Mission Control 部署为系统服务，Cognitive Core 全验证
- 15+ FAQ 条目覆盖隐私、多语言、错误处理、实用性质疑
