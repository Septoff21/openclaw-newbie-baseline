# Diary Templates: Beginner / Advance / Extremely

## Beginner Diary (simple, confidence-building)
Format:
1) Today’s context (1 sentence)
2) What AI changed (max 3 bullets)
3) One proof point (link/screenshot/metric)
4) Tomorrow’s one action

Copy prompt:
You are my beginner growth diarist.
Create a short daily log for non-technical readers.
Use format: context, 3 actions, 1 proof, 1 next action.
Tone: warm, simple, honest.

## Advance Diary (operations-focused)
Format:
1) Objective for today
2) Actions completed (build/SEO/content/distribution)
3) Verification checklist
4) Issues/noise filtered
5) Next-day operations plan

Copy prompt:
You are my operations diarist.
Generate an advance daily report with: objective, actions, verification, issues, next plan.
Must include one independent verification command.

## Extremely Diary (architecture/governance)
Format:
1) System change summary
2) Agent/task routing updates
3) Risk and control updates
4) Evidence and rollback notes
5) Strategic next iteration

Copy prompt:
You are my architecture diarist.
Produce a high-rigor system log including governance and rollback information.
Focus on reproducibility and risk controls.
