Quick Start Guide — 快速开始

📦 Before You Start / 开始之前

You need these 3 things (that's all): / 你只需要这 3 样东西：

  1. A computer (Mac, Windows, or Linux) / 一台电脑（Mac、Windows 或 Linux）
  2. 5 minutes of uninterrupted time / 5 分钟不被打断的时间
  3. A free API key from openrouter.ai/keys (we'll show you how) / openrouter.ai/keys 的免费 API 密钥（我们会教你）

Don't know what a terminal is? On Mac: press Cmd+Space, type "Terminal", press Enter. On Windows: press Win+R, type "powershell", press Enter. That's it! / 不知道终端是什么？ Mac：按 Cmd+Space，输入"Terminal"，按回车。Windows：按 Win+R，输入"powershell"，按回车。

Step 1: Install OpenClaw / 安装 OpenClaw ⏱ 2 min

Open your terminal (Terminal on Mac/Linux, PowerShell on Windows) and paste this:
打开终端（Mac/Linux 用 Terminal，Windows 用 PowerShell）并粘贴：

npm install -g openclaw && openclaw doctor

Just copy the full line above and press Enter — no editing needed for first run.
直接复制上面整行后按回车即可，第一次不用改任何内容。

This installs OpenClaw and checks your system. If you see ✅, you're good.
这会安装 OpenClaw 并检查系统。如果看到 ✅ 就没问题。

🔧 Troubleshooting / 常见问题
  • "command not found" — Install Node.js first from nodejs.org
  • Permission denied — Try sudo npm install -g openclaw
  • Windows errors — Run PowerShell as Administrator

Step 2: Configure / 配置 ⏱ 1 min

Set up your API key. We recommend OpenRouter (one key for all models):
设置 API 密钥。推荐 OpenRouter（一个密钥用所有模型）：

openclaw config set model.provider openrouter
openclaw config set model.apiKey "your-api-key-here"

Get a free API key at openrouter.ai/keys 🔑
免费获取密钥：openrouter.ai/keys

Step 3: Give It a Soul / 给它一个灵魂 ⏱ 1 min

Copy this prompt and paste it into your AI chat. This sets up your agent's personality:
复制此提示词并粘贴到 AI 聊天中。这会设置你 agent 的人格：

Set up my OpenClaw agent with these files:

SOUL.md:
- Name: [Your choice]
- Vibe: Helpful but not sycophantic
- Boundaries: Private data stays private, ask before external actions
- Style: Concise when needed, thorough when it matters

AGENTS.md:
- Read memory files each session
- Daily notes in memory/YYYY-MM-DD.md
- Long-term memory in MEMORY.md
- Don't exfiltrate private data
- trash > rm (recoverable beats gone forever)

USER.md:
- Name: [Your name]
- Timezone: [Your timezone]
- Preferences: [Your preferences]

Create these files and show me the result.

Step 4: Verify / 验证 ⏱ 30 sec

Run these commands to make sure everything works:
运行这些命令确保一切正常：

openclaw status        # Should show "running"
openclaw chat "Hello"  # Should get a response

If both work → 🎉 your agent is alive!
如果两个都工作 → 🎉 你的 agent 活了！

Step 5: Connect / 连接 ⏱ 1 min

Connect your agent to a messaging platform:
将你的 agent 连接到消息平台：

📱 Telegram

openclaw connect telegram

💬 WhatsApp

openclaw connect whatsapp

🎮 Discord

openclaw connect discord

🍎 iMessage

openclaw connect imessage

💡 First Week Tips / 第一周小贴士

  • Day 1–2: Just talk to it. Ask random questions, give it small tasks, see how it responds. Don't overthink setup yet.
  • Day 3–4: Edit SOUL.md based on what annoyed you. "Stop apologizing" or "be more direct" — make it yours.
  • Day 5–7: Give it a real project. Something small — organize files, draft emails, research a topic. Real tasks = real feedback.
  • Track costs: Check your API dashboard after a few days. Most people are surprised how cheap it is — or how fast tokens burn with verbose agents.
  • Memory is your superpower: The agent remembers what you write in MEMORY.md. The more you invest in memory, the smarter it gets over time.
  • Don't fear resets: Messed up SOUL.md? Wipe it and start fresh. Nothing breaks. It's just text files. / 不怕重来：SOUL.md 搞砸了？清空重来就好。不会出错的，只是文本文件。
