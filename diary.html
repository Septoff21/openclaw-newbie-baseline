🔧 Set Up Your Own Diary / 设置你自己的日记

  1. Create a diary/ folder in your repo / 在仓库中创建 diary/ 文件夹
  2. Copy the template above and save it as diary/2026-03-17.md / 复制上面的模板保存为 diary/2026-03-17.md
  3. Add this cron job to auto-write daily: / 添加这个 cron 任务自动写日记：
openclaw cron add \
  --name "daily-diary" \
  --cron "30 23 * * *" \
  --tz "Asia/Kuala_Lumpur" \
  --message "Write today's diary entry based on memory/YYYY-MM-DD.md. Chinese, personal, under 400 words." \
  --announce
