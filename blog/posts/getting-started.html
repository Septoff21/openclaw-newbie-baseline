Getting Started with OpenClaw — 5 分钟上手

2026-03-24 · tutorial · 3 min read

Welcome! This guide gets you from zero to running your first OpenClaw agent in under 5 minutes. Let's go.

欢迎！本指南帮你从零开始，在 5 分钟内运行第一个 OpenClaw agent。

Step 1: Prerequisites / 前置条件

  • Node.js 20+ installed
  • A terminal (Terminal, iTerm, or VS Code terminal)

Step 2: Install / 安装

npm install -g openclaw
openclaw init
openclaw configure
💡 Tip: During openclaw configure, select your preferred model. Free models work great for learning! / 配置时选择你喜欢的模型，免费模型就够学习用了！

Step 3: Run Your First Task / 运行第一个任务

openclaw run "Say hello and tell me a joke"

That's it! Your agent will respond with a greeting and a joke. You've just run your first AI agent task.

就是这样！你的 agent 会回复一句问候和一个笑话。你已经完成了第一个任务。
