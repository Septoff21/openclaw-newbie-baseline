ClawHub：你的第一个 AI Agent 技能市场

2026-03-24 · skills · marketplace · ClawHub · 8 min read

AI Agent 缺的不是大脑，是技能

大语言模型已经足够聪明——它能理解意图、推理问题、生成代码。但当你要它"帮我查一下邮件""把这个 PR review 一下""监控服务器状态"时，光有大脑不够，它还需要手和脚。

这些"手和脚"就是 Agent 技能（Skills）——预打包的能力模块，让 AI Agent 知道该用什么工具、怎么用、什么时候用。

ClawHub 就是答案。

ClawHub 是什么？

一句话定义：ClawHub 是 AI Agent 的 npm。

# 搜索技能
clawhub search "github"

# 安装技能
clawhub install github

# 更新所有技能
clawhub update --all

# 发布你自己的技能
clawhub publish ./my-skill

为什么需要 ClawHub？

问题一：Agent 能力的"手动挡时代"

在没有技能市场之前，给 AI Agent 添加能力需要：手动搜索 GitHub、克隆仓库、复制文件、修改配置、祈祷兼容。对普通用户来说是不可能完成的任务。

问题二：能力孤岛

  • LangChain Tools：Python 生态，绑死 LangChain
  • AutoGen Plugins：微软体系
  • Dify Tools：SaaS 平台，私有格式
  • OpenAI GPTs：完全封闭

问题三：质量参差不齐

社区分享的工具散落各处，没有统一标准：没文档、有安全漏洞、依赖过时……

ClawHub 的解决方案

  1. 标准化：技能遵循统一的 SKILL.md 规范，跨框架兼容
  2. 集中化：一个市场，搜索即所得
  3. 质量门控：发布前的安全审查和元数据验证

核心概念

Skill（技能）

ClawHub 管理的基本单位是 Skill，一个自包含的能力包：

my-skill/
├── SKILL.md          # 技能定义文件（必选）
├── scripts/          # 辅助脚本（可选）
├── references/       # 参考文档（可选）
└── prompts/          # 提示模板（可选）

SKILL.md 解剖

SKILL.md 是灵魂，它告诉 AI Agent：这个技能是干什么的、什么时候触发、怎么使用、有什么注意事项。

Skill 类型

类型说明示例
工具型封装 CLI 工具或 APIgithub、docker、k8s
工作流型多步骤自动化流程deploy-pipeline、code-review
知识型领域知识和最佳实践security-checklist、seo-guide
MCP Server 型封装 MCP Server 配置mcp-postgres、mcp-filesystem

快速上手

安装 ClawHub CLI

# 通过 npm 全局安装
npm install -g clawhub

# 验证安装
clawhub --version

搜索 & 安装技能

# 按关键词搜索
clawhub search "code review"

# 安装技能
clawhub install github

# 安装特定版本
clawhub install github@2.1.0

管理已安装的技能

# 列出已安装技能
clawhub list

# 检查更新
clawhub outdated

# 更新所有技能
clawhub update --all

# 卸载技能
clawhub uninstall github

深入：技能架构

技能发现机制

  1. 目录扫描：Agent 启动时扫描 skills/ 目录下的所有 SKILL.md
  2. 元数据提取：解析 Description 和 Trigger 字段
  3. 上下文匹配：Agent 将用户指令与技能 Trigger 做语义匹配
  4. 加载执行：匹配成功后按 Usage 指引执行

这个机制的优雅之处在于：技能的触发完全由 AI 自主决定，不需要用户手动指定。

发布你的技能

# 1. 初始化技能目录
clawhub init my-awesome-skill

# 2. 本地验证
clawhub validate

# 3. 登录并发布
clawhub login
clawhub publish

安全审查

  • 静态分析：扫描危险命令（rm -rf、curl | bash 等）
  • 依赖检查：检查已知漏洞
  • 权限审计：确保只申请必要权限
  • 人工抽查：热门技能会被人工 review

常见问题

Q：ClawHub 和 MCP 是什么关系？

A：互补关系。MCP 定义了 AI Agent 如何连接工具的协议标准，ClawHub 定义了技能如何分发和管理的市场标准。

Q：技能会不会有安全风险？

A：建议只安装有足够下载量和好评的技能，安装前阅读 SKILL.md，在沙箱环境先测试。

总结

ClawHub 不是一个产品，而是一个生态基础设施。它让 AI Agent 的能力可以像软件包一样被搜索、安装、分享和管理。
