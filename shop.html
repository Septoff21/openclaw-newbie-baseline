✅ Complete Example: Build a "Daily Weather" Skill in 10 Minutes / 完整案例：10 分钟做一个"每日天气"技能

Follow these steps. Commands are examples — check docs for exact syntax.
跟着这些步骤。命令是示例——查看 文档 获取确切语法。

Step 1: Create the skill file / 创建技能文件

Create a file called skills/weather-summary/SKILL.md in your OpenClaw folder:

# SKILL.md — Weather Summary Skill

## What it does
Summarizes today's weather in a friendly one-liner.

## How to use
Say: "What's the weather like today in [your city]?"

## Script
Runs `curl wttr.in/{city}?format=3` and returns the result.

## Monetization
Free version: 1 city. Paid ($2/mo): unlimited cities + alerts.

Step 2: Test it locally / 本地测试

openclaw skills test weather-summary

Step 3: Publish to ClawHub / 发布到 ClawHub

clawhub publish skills/weather-summary --price free

Step 4: Share / 分享

Your skill is now live! Share the link: clawhub.com/skills/your-username/weather-summary
