🧠 7 Ways to Save Money / 7 个省钱方法

  1. Start with free models — Hunter Alpha handles 90% of tasks fine
    从免费模型开始——Hunter Alpha 能处理 90% 的任务
  2. Use context warnings — Don't let models forget and waste tokens
    使用上下文警告——别让模型忘记东西浪费 token
  3. Summarize before switching — Compress context before using expensive models
    切换前总结——用贵模型之前先压缩上下文
  4. Batch similar tasks — One long prompt > ten short ones
    批处理相似任务——一个长提示 > 十个短提示
  5. Set daily limits — OpenRouter lets you cap spending
    设置每日限额——OpenRouter 可以限制支出
  6. Cache common responses — Don't re-ask the same questions
    缓存常见回答——别重复问同样的问题
  7. Monitor usage weekly — Check your OpenRouter dashboard
    每周监控用量——查看 OpenRouter 仪表盘

# OpenClaw cost-optimized config (example / 示例)

# Default: free model for daily tasks
openclaw config set model.default openrouter/hunter-alpha

# Fallback: cheap model for complex tasks
openclaw config set model.fallback openrouter/gpt-5.4-mini

# Emergency: premium for critical decisions only
openclaw config set model.premium openrouter/gpt-5.4

# Set daily spending limit ($1/day max)
openclaw config set billing.dailyLimit 1.00

# Enable context warnings (auto-compress at 70%)
openclaw config set context.warningThreshold 70
openclaw config set context.autoCompress true
